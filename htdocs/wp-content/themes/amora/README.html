Description

Amora is a unique, ultra creative multipurpose WordPress theme. Ranker supports all basic features and is fully mobile friendly. And with Support for Custom Widgets, Featured Posts section, etc you can make this theme look even more beautiful on your website. Demo Here: http://demo.inkhive.com/amora/
